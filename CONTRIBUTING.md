# Contribuer au projet Casino Web

Merci de votre intérêt pour contribuer !  

## Règles générales
- Respectez les bonnes pratiques HTML, CSS et JS.
- Commentez tout code nouveau ou modifié.
- Testez les modifications avant de proposer un PR.

## Étapes pour contribuer
1. Forkez le dépôt.
2. Créez une branche : `git checkout -b feature/ma-fonctionnalité`
3. Faites vos modifications et commit avec messages clairs.
4. Poussez votre branche et ouvrez un Pull Request.

## Bonnes pratiques
- Ajouter des tests pour les calculs de gains ou animations critiques.
- Documenter toute nouvelle fonction ou module dans `ARCHITECTURE.md`.
- Vérifier que l'interface reste responsive et fonctionnelle sur tous les jeux.
