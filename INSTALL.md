# Installation de Casino Web

## Prérequis
- Navigateur web moderne (Chrome, Firefox, Edge, Safari)
- Aucun serveur requis (fonctionne en local)

## Étapes
1. Télécharger le projet ou cloner le dépôt :
```bash
git clone https://github.com/Rafael-Iste06/Web-Casino-Online-Games
cd web_casino_online_games
```
2. Ouvrir index.html dans votre navigateur préféré.
3. Commencer à jouer immédiatement.

## Notes
* Les jeux utilisent uniquement des technologies web standard.
* Les animations sont optimisées pour tous les navigateurs modernes.
